| Layer | Technology |
|-------|-----------|
| Structure | HTML5 Semantic Markup |

## ✨ Features

- 🎯 Giant hero background wordmark with orange brand accent
- 🍔 Animated burger menu with slide-in nav drawer
- 📱 Fully responsive — mobile, tablet, desktop
- 🌊 Scroll-reveal animations with IntersectionObserver
- 🖱️ Cursor glow effect (desktop only)
- 🔢 Animated number counters
- ♿ Accessible — ARIA labels, focus management, keyboard navigation
- ⚡ Zero dependencies — instant load
